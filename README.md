A simple web application using python integrated with PostgreSQL. 
1. The application has a basic CRUD(Create, Read, Update , delete) functionality via ReST API
2. The application has an one API Integration that is Gemini API from which we will send the data.
3. A simple data visualization, showcasing data retreiving from the database i.e,  the spending of your finances like rent, food, personal, health etc.
## Run Locally
**Prerequisites:**  Node.js
1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key, replace the API key with your API key in .env file. 
3. Run the app:
   `npm run dev`
